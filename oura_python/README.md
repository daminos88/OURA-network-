# OURA Python Runtime

This package is the Python runtime mirror for OURA.

Planned modules:
- `types.py`
- `errors.py`
- `verifier.py`
- `evaluator.py`
- `engine.py`

Target capabilities:
- parse trust stores
- load policy bundles
- verify bundle structure
- perform signature verification (placeholder until full crypto lands)
- evaluate rules deterministically
- emit parity artifacts comparable to Dart
